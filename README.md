# Competitive Programming

A collection of competitive programming solutions, templates, algorithms, and problem-solving notes in C++.

This repository documents my journey toward mastering competitive programming and problem solving. It covers frequently tested topics in contests like Codeforces, AtCoder, and ICPC.

## Topics Covered

- Data Structures: DSU (Disjoint Set Union), Trie, Segment Tree, Sqrt Decomposition
- Graph Algorithms: DFS, BFS, Dijkstra, MST/Kruskal
- Dynamic Programming: LIS (Longest Increasing Subsequence), CIS (Common Increasing Subsequence)
- Number Theory: Sieve of Eratosthenes, Prime checking, Binary Exponentiation
- Techniques: Prefix Sum, 2D Prefix Sum, Bitmask Subsets, Frequency Array
- Contest Templates (ready-to-use starting points)

## Technologies Used

- C++
- Standard Template Library (STL)
- Algorithms and Data Structures

## Installation

### Prerequisites

- C++17 compatible compiler (g++, clang++)

### Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Obada111/competitive-programming.git
   cd competitive-programming
   ```

2. Browse topics:
   ```bash
   ls topics/
   ls competitive-programming/
   ```

3. Compile and run any algorithm:
   ```bash
   g++ -std=c++17 topics/segment-tree.cpp -o segtree
   ./segtree
   ```
